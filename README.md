## master
マスターブランチです。正式公開されたバージョンをここにいれます。



## develop
開発中のブランチです。


基本はこのブランチを追いかけてください

このブランチをチェックアウト後、自分のブランチを作成して作業してください。
ブランチ名は
MCID/ブランチ名

[例] takatronix/xxxx
